# react-native-smile-id

Official wrapper for the Smile ID <v10 android and iOS SDKs

## Installation

```sh
npm install react-native-smile-id
```

## Usage

```js
import { multiply } from 'react-native-smile-id';

// ...

const result = await multiply(3, 7);
```
